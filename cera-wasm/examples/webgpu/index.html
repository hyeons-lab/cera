cera-wasm · WebGPU LFM2 text generation

Loads an LFM2 GGUF entirely in your browser and runs prefill + greedy decode on the GPU via WebGPU (no server inference). Requires a WebGPU-capable browser (Chrome/Edge 113+, or Safari Technology Preview). A real model is hundreds of MB — this page is the manual end-to-end harness; the automated test (webgpu_smoke.rs) covers the async GPU host layer.
